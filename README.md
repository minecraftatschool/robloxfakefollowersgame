# robloxfakefollowersgame
robloxfakefollowersgame is a Roblox game where you earn fake followers.
You can use Rojo to build the game, or use the version of Rojo provided and run this in the terminal:
# Powershell
```
./rojo.exe build -o build.rbxl
```
# Command Prompt
```
rojo build -o build.rbxl
```
